Let $q$ be the position of object 1, and $r$ be the position of object 2 and $v_q$, $v_r$ be their respective velocities.
Reframe the problem such that $p = q - r$, $v = v_q - v_r$. Now it's a one object problem w.r.t the origin.

Distance from the origin is
$ d = \lVert p + v t \rVert = \sqrt{\left( p_x + v_x t \right)^2 + \left(p_y + v_y t\right)^2}$
$ d = \sqrt{t^2\left(v_x^2 + v_y^2\right) + t(2p_x v_x + 2p_y v_y) + (p_x^2 + p_y^2)}$
$ \frac{dd}{dt} = \frac{t(v_x^2+v_y^2) + (p_x v_x+p_y v_y)}{ \sqrt{t^2\left(v_x^2 + v_y^2\right) + t(2p_x v_x + 2p_y v_y) + (p_x^2 + p_y^2)}}$

$\frac{dd}{dt}$ minimizes at $t = \frac{-(p_x v_x+p_y v_y)}{v_x^2+v_y^2}$
